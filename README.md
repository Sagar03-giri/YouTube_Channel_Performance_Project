# Unlocking YouTube Channel Performance Secrets

## Project Overview

This project focuses on analyzing YouTube channel performance data to uncover growth patterns, engagement factors, revenue drivers, and audience behavior. The project combines Exploratory Data Analysis (EDA), Business Intelligence, and Machine Learning techniques to generate meaningful insights and support data-driven content strategies.

The analysis was performed using Python for data analytics and Power BI for interactive dashboard visualization.

---

# Objectives

- Analyze YouTube performance metrics
- Identify factors affecting channel growth
- Understand monetization and revenue trends
- Discover audience engagement patterns
- Build predictive insights using Machine Learning
- Create an interactive Power BI dashboard

# Dataset Features

The dataset contains various YouTube performance metrics such as:

- Views
- Watch Time
- Likes
- Shares
- Subscribers
- CTR (Click Through Rate)
- Revenue
- Impressions
- Monetized Playbacks
- CPM
- Video Duration
- Publishing Date
- Engagement Metrics

---

# Project Workflow

## 1. Data Cleaning
- Removed duplicates
- Handled missing values
- Fixed inconsistent data formats

## 2. Feature Engineering
Created new metrics such as:
- Engagement Rate
- Revenue per Watch Hour
- Subscriber Conversion Rate

## 3. Exploratory Data Analysis (EDA)
Performed:
- Distribution analysis
- Correlation analysis
- Revenue trend analysis
- Engagement analysis
- Revenue source analysis

## 4. Machine Learning
Built a Random Forest Regressor model to predict YouTube revenue using:
- Views
- Watch Time
- CTR
- Impressions
- Engagement Metrics

---

# Key Insights

- Watch Time strongly impacts revenue generation.
- Higher CTR contributes to better monetization.
- AdSense is the dominant revenue source.
- Engagement metrics positively influence subscriber growth.
- Certain publishing days generate higher revenue performance.
